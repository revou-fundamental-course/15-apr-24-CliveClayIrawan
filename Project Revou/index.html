How do you convert (°F) to (°C)?

To convert Celsius to Fahrenheit, multiply by 9/5 (or 1.8) and then add 32. For example, 0 degrees Celsius is equal to (0 * 9/5) + 32 = 32 degrees Fahrenheit.

S(°F) = (S(°C) x 9/5) + 32

Or

S(°F) = (S(°C) x 1.8) + 32
